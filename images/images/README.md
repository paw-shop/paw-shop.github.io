# Carpeta de imagenes

Pon aqui las imagenes de tus proyectos.

## Estructura sugerida

- `images/project-1/cover.jpg`
- `images/project-1/render-1.jpg`
- `images/project-1/render-2.jpg`
- `images/project-2/cover.jpg`
- `images/project-2/render-1.jpg`
- `images/project-3/cover.jpg`

## Como usarlo en el editor o en projects.json

Imagen principal:

`./images/project-1/cover.jpg`

Galeria:

`./images/project-1/render-1.jpg`
`./images/project-1/render-2.jpg`
`./images/project-1/render-3.jpg`

Usa nombres simples y evita espacios.
