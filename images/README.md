# Carpeta de imagenes

Pon aqui las imagenes de tus proyectos.

## Estructura sugerida

- `images/project-1/proyect1.1.jpg`
- `images/project-1/proyect1.2.jpg`
- `images/project-1/proyect1.3.jpg`
- `images/project-2/proyect2.1.jpg`
- `images/project-3/proyect3.1.jpg`
- `images/project-7/project-7.1.jpg`

## Como usarlo en el editor o en projects.json

Imagen principal:

`./images/project-1/proyect1.1.jpg`

Galeria:

`./images/project-1/proyect1.1.jpg`
`./images/project-1/proyect1.2.jpg`
`./images/project-1/proyect1.3.jpg`

Usa nombres simples y evita espacios.

Este repo ya trae ejemplos de imagenes dentro de `images/project-1/` a `images/project-7/`. Puedes reemplazarlos o agregar los tuyos con la misma estructura.
